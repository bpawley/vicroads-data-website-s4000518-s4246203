Injury Risk Deep Dive

"Discover which groups have above-average serious or fatal injury outcomes."

Home › People & Injuries › Deep Analysis
How This Deep Dive Works
Query Step 1

Calculate the overall serious/fatal injury rate for selected records.
SELECT ROUND(SUM(CASE WHEN INJ_LEVEL IN (1,2) THEN 1.0 ELSE 0 END) * 100.0 / COUNT(*), 2) FROM Person [WHERE filters]

Query Step 2

Group records by road user type, age group, hospitalisation status, and ejection status. Calculate each group's serious/fatal injury rate.
SELECT road_user_type, age_group, COUNT(*), serious_fatal_rate FROM Person JOIN Road_User ... GROUP BY ...

Query Step 3

Compare each group's serious/fatal injury rate against the overall average from Step 1.
HAVING rate > [overall_rate from Step 1]

Output

Display groups with above-average injury outcomes, ranked by difference from average.
(Level 3 functions use the result of one query as input to the next.)
